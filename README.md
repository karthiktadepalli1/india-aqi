# india-aqi

Supporting repository for my [analysis](https://www.karthiktadepalli.com/covid-pollution/) of how the COVID-19 lockdown has impacted India's air quality. This repository consists of:

1. The code for a [Shiny app](https://karthik-tadepalli.shinyapps.io/aqi-breaks/) simulating arbitrary breaks to compare the real lockdown's effect to a synthetic lockdown effect.
2. The code to scrape India's AQI bulletin for recent AQI values in major Indian cities.
